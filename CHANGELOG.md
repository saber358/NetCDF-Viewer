# 更新日志

本文件记录项目的主要版本变化。

## 1.1.1

- 新增风场叠加，自动识别三角网 `uwind_speed` / `vwindspeed`、`uwind_speed` / `vwind_speed` 以及规则格网 `U10` / `V10`
- 风场叠加采用风羽标记绘制，三角网和规则格网共用同一控制入口与独立开关
- 新增规则格网向量波场支持，存在 `uWave` / `vWave` 时即可渲染浪场，若同时存在 `Hwave` / `hWave` 则用于增强箭头长度表达
- 补充风场变量识别、风羽采样、主控制器开关联动与规则格网 / 三角网回归测试
- 更新 `README`，重写项目能力说明，补充规则格网标量、海流、浪场、风场与三角网风场实际截图
- 重新打包 Windows 安装程序，生成 `NetCDFViewer-1.1.1.exe`
- 版本号升级到 `1.1.1`

## 1.1.0

- 新增标准格网 NetCDF 支持，可加载规则格网类型数据，不再局限于非结构化三角网
- 新增统一空间域模型，三角网与标准格网共用主控制链路、视口适配和渲染入口
- 新增标准格网标量渲染，支持规则一维横纵坐标的节点场与单元场二维平面显示
- 新增标准格网单点查询，点击后可返回命中节点或单元的行列位置与数值
- 新增标准格网坐标基准切换，标准格网数据可按当前变量自动匹配兼容横纵坐标，并允许用户切换候选坐标轴
- 新增标准格网向量采样，支持同基准和交错基准的 `u` / `v` 结构化流场联合采样
- 新增标准格网海流流线叠加，支持结构化 ROMS / WRF 类速度场渲染动态流线
- 新增标准格网波向箭头采样，为后续结构化波场叠加提供统一采样基础
- 新增结构化渲染、点查、坐标控件、向量采样、流线与控制器回归测试，并完成 Maven 全量测试通过
- 版本号升级到 `1.1.0`

## 1.0.5

- 新增波浪波向叠加，自动识别兼容的 `wdir` / `wlen` 变量对，并在平面图上绘制波向箭头
- 新增海流流场叠加，自动识别兼容的 `u` / `v` 或 `ua` / `va` 变量对，并以动态黑色短流线显示流向
- 优化海流流线观感，缩短单段笔触并增加路径平滑处理，减少转折生硬和动画抖动
- 新增波向叠加与海流流线叠加相关识别、渲染和回归测试
- 更新 GitHub / Gitee `README`，补充最近新增功能说明与波向、海流预览截图
- 版本号升级到 `1.0.5`

## 1.0.4

- 新增 GeoJSON / shapefile 海岸线叠加，可作为参考边界覆盖到当前平面图之上
- 海岸线叠加跟随当前视图缩放、平移和数据集切换一起刷新
- 新增内置默认海岸线资源，程序启动后默认启用，无需用户手动加载文件
- 新增 `Use Built-in Coastline` 菜单入口，可在清空或替换外部海岸线后恢复系统内置海岸线
- 新增本地海岸线样例数据：`samples/coastline/ne_110m_coastline.geojson` 与 `samples/coastline/ne_110m_coastline.shp`
- 新增真实海岸线样例解析回归测试，验证下载到本地的 GeoJSON 和 SHP 文件可被成功读取
- 重新打包 Windows 安装程序，更新 `NetCDFViewer-1.0.4.exe`
- 版本号升级到 `1.0.4`
