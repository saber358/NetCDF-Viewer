# 更新日志

本文件记录项目的主要版本变化。

## 1.0.2

- 修复 PNG 导出链路的健壮性问题
- 新增导出后文件可读回校验，避免静默生成损坏图片
- 新增打包运行时导出回归测试
- 新增平面图单点查询，支持鼠标点击查询当前点对应的网格值
- 单元中心变量返回命中三角形原始值，节点中心变量返回三角形内插值结果
- 新增单点查询交互与几何计算回归测试
- 重新打包 Windows 安装程序，更新 `NetCDFViewer-1.0.2.exe`
- 版本号升级到 `1.0.2`

## 1.0.1

- 新增作者信息展示：`lwj / 2762692204@qq.com`

- 更换应用与安装包图标
- 更新安装包元数据与版本号

## 1.0.0

- 完成 NetCDF 非结构化三角网桌面可视化基础能力
- 支持节点 / 单元属性变量平面渲染
- 支持多层变量深度切换
- 支持 PNG 导出
- 支持 Windows `.exe` 打包
